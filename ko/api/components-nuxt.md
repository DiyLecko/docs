# &lt;nuxt&gt; 컴포넌트

> 이 컴포넌트는 페이지 컴포넌트가 있는 [layouts](/guide/views#layouts)에서만 사용됩니다.

예제 (`layouts/default.vue`):

```html
<template>
  <div>
    <div>My nav bar</div>
    <nuxt/>
    <div>My footer</div>
  </div>
</template>
```

[layouts example](/examples/layouts)에서 예제를 보실 수 있습니다.
